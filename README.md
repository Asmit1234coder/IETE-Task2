# Week 2 - Task 2

Welcome to **Week 2 - Task 2**.

## Objective

Complete the assigned task by following the instructions below and submit your solution through a Pull Request.

---

## Steps to Follow

### 1. Clone the Repository

```bash
git clone <repository-url>
cd <repository-name>
```

### 2. Create Your Branch

Use your name as the branch name.

```bash
git checkout -b your-name
```

Example:

```bash
git checkout -b asmit
```

### 3. Complete the Task

- Write clean and readable code.
- Follow proper folder structure.
- Test your solution before submitting.

### 4. Commit Your Changes

```bash
git add .
git commit -m "Completed Week 2 Task 2"
```

### 5. Push Your Branch

```bash
git push origin your-name
```

Example:

```bash
git push origin asmit
```

### 6. Create a Pull Request

- Open the GitHub repository.
- Click **Compare & Pull Request**.
- Set the **base branch** as `main`.
- Add a meaningful title and description.
- Submit the Pull Request.

---

## Rules

- ❌ Do not push directly to the `main` branch.
- ✅ Work only on your own branch.
- ✅ Make meaningful commit messages.
- ✅ Ensure your code compiles/runs before creating the PR.
- ✅ Do not modify other participants' files.

---

- [ ] Changes committed
